# Open-Source Recipes 

### The Mission: No Ads. No Stories. Just Food.

I built this repository because I was tired of digging through commercial recipe websites filled with:
* ‚ùå Intrusive popup ads and trackers.
* ‚ùå 2,000-word "backstories" about the author's childhood or how much their family loves this recipe.
* ‚ùå Unnecessary cooking theory bloat.

**This repo provides:**
1.  **A picture** of the final "build."
2.  **A straightforward description** of the product.
3.  **A shopping list** categorized for a quick grocery run.
4.  **To the point Instructions**: Precise, step-by-step "source code" for your kitchen.

## üõ† How to Use This Repo
* **Fork it**: Want to tweak the salt levels or swap chocolate for nuts? Fork the repo and maintain your own "branch."
* **Print it**: Every recipe includes a placeholder for a 3x5 index card graphic for your physical kitchen files.
* **Contribute**: If you've found a "bug fix" for a recipe (like a better oven temp for high altitudes), feel free to open a Pull Request.

---

## üí° Core Technical Standards
All recipes in this repository assume the following "Global Config" unless otherwise stated:
* **Aerate your flour**: Never scoop from the bag.
* **Temper your butter**: Softened, never soupy.
* **Precision Leveling**: Use the flat edge of a butter knife for all dry measurements.
